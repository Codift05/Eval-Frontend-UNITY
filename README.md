## Overview

This platform provides a centralized, secure interface to systematically evaluate frontend student portfolios based on a comprehensive rubric. It focuses on delivering detailed, action-oriented feedback in a professional and accessible format.

## Key Features

* **Secure Authentication**
  Students access their private, detailed score breakdowns and personalized feedback via unique credential codes.
* **Global Leaderboard**
  A dynamic ranking system displaying top-performing projects to encourage healthy competition and transparency.
* **Modern Interface**
  A premium, minimalist dark-mode aesthetic utilizing clean typography (Inter) and subtle architectural borders for maximum readability.
* **Visual Data Representation**
  Metrics are presented through responsive, animated score bars for immediate cognitive processing.

## Evaluation Criteria

The system assesses projects based on the following metrics:
1. **UI/UX & Styling** - Interface aesthetics, color theory, and user experience.
2. **Responsivitas** - Fluidity and adaptability across mobile, tablet, and desktop devices.
3. **Fungsionalitas & State Management** - Core logic, data handling, and interactive components.
4. **Struktur Komponen & Clean Code** - Code architecture, modularity, and maintainability.
5. **Kreativitas & Extra Effort** - Additional polish, micro-animations, and innovative solutions.

## Getting Started

### Prerequisites

* Node.js (version 18 or higher)
* Package manager (npm, yarn, or pnpm)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Codift05/Eval-Frontend-UNITY.git
   ```

2. Navigate into the directory
   ```bash
   cd Eval-Frontend-UNITY
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

## Architecture

* **Framework**: React.js
* **Build Tool**: Vite
* **Styling**: Vanilla CSS (CSS Variables)
* **Icons**: Lucide React
* **Data Layer**: Static Mock Data (Configured for future Backend-as-a-Service integration)
